## CloudIQ Tech Test

A working demo of this app can be found on Heroku at http://cloud-iq-test.herokuapp.com/

To get a local version:
```
git clone https://github.com/sensationmm/cloud-iq.git
cd into directory
npm install
npm start
```

To run the test suite:
```
npm test
```
